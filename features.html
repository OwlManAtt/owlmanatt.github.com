--- layout: default title: Features --- Kitto provides you with the following out-of-the-box:
  • User Management
    • Registration (with CAPTCHA)
    • Login/logout/user preferences
    • Password reset
    • Permission groups (for staff powers)
    • Admin panel for editing/banning users
    • Search
    • Timezone & date/time format localization options
  • Forums
    • Avatars, signatures, post counts, and user titles
    • Locked-down 'News' boards - only admins can post here, and each thread shows up as a news item. Subsequent posts in a news thread show as comments!
    • Sticky/locked threads
    • Delete/Edit posts and threads
    • Full power of HTML can be utilized for post mark-up - safely. A rich text editor is available to make adding flair to your post easy
  • Private Messaging with multiple recipients for a message
  • Event notification (with bulk event deletion)
  • Pet Management
    • Adopt a member of a pet species in several colours
    • Locked-down colours available only via a paintbrush (or whatever means you'd like)
    • Hunger/happiness levels
    • Pet profiles
    • Abandonment
  • Inventory system
    • Stacking items (6 Red Apples = one item entry)
    • Four item types - food, toys, paintbrushes, and recipes
    • Crafting - create new items with recipes and ingredients
    • NPC shops with restocks
